# Demo Path

This is the shortest honest demo path for the repository.

## Five-Minute Sequence

1. Show `make verify-release` as the single release-grade command.
2. Open `reports/release/release_readiness_report.md`.
3. Hit `/health` and `/config`.
4. Call `/score/backbones` with a small backbone list.
5. Call `/graphql` with the same backbone IDs to show the compact query surface.
6. Show `reports/headline_validation_summary.md` and `reports/core_tables/blocked_holdout_summary.tsv`.

## Narrative Rule

- Lead with retrospective surveillance scope.
- Say explicitly that this is not a clinical decision surface.
- Tie every claim to a generated artifact or contract document.
