# Investor Overview

The commercial wedge is not generic prediction infrastructure. It is governed retrospective surveillance prioritization with a strong evidence and reproducibility layer.

## Why This Matters

- operational users need prioritization, not raw model scores
- scientific credibility is part of the product surface
- release discipline reduces trust erosion

## What Improves Distribution

- a small API surface
- one-command verification
- stable demo path
- stakeholder-specific documentation that does not overclaim
