# Scientific Overview

Plasmid Priority is a retrospective genomic surveillance framework for operational plasmid backbone classes. It scores backbones across `geo_spread`, `bio_transfer`, and `clinical_hazard`, then fuses them into a guarded consensus surface.

## Scientific Posture

- retrospective labels only
- temporal leakage controls
- generated reviewer-pack artifacts
- release claims constrained by contract and invariant gates

## What Counts As Credible Here

- reproducible generated artifacts
- stable release protocol documents
- explicit claim-level boundaries
- invariant-aware comparisons against frozen baselines
