# Experimental Artifacts

This directory is reserved for model-search outputs, exploratory sweeps, and other non-canonical experiment artifacts.

Rules:

- `data/analysis/` remains the canonical machine-readable analysis surface.
- Experimental search outputs belong here, not under `data/analysis/`.
- `registry.tsv` is generated and records checksums, sizes, and timestamps for the experimental files currently present.
- Files in this directory are not part of the headline scientific report unless explicitly promoted later.
